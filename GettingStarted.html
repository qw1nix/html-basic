Getting started

System requirements

  • (Windows users) WSL2 (Windows Subsystem for Linux). We use Ubuntu 20.04 and you need to use version 2 of WSL.
  • Docker. Get it for your system from here.If you use Windows - please, use WSL 2 based engine for Docker.

General prerequisites

  1. Install nvm(macOS/Linux, Windows).
  2. WARN: please, use lts version of node 16. , currently (05.05.22) - lts/gallium* Install lts node version. Run nvm install lts
  3. Run nvm use lts to use it.

Setup

  1. Clone repo
  2. Install packages. Run npm i
  3. Add cypress.env.json file to the root of project with following format:
 {
   "USERNAME": "value", 
   "PASSWORD": "value"
 }
